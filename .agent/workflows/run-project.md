description: How to run and test the AI Pull Request Reviewer
---

1. Ensure you have Node.js installed.
2. Open the terminal in the project root.
// turbo
3. Run `npm install` to install dependencies.
// turbo
4. Run `npm run dev` to start the development server.
5. Open the provided local URL in your browser to experience the multi-billion dollar UI.
6. Trigger the "Run Analysis" button to simulate a codebase scan.
7. Click on any card (e.g., "Circular Dependencies") to view the detailed intelligence panel.
